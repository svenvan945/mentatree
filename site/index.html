Install for Claude Code See how it works

The advice in great books rots as highlights.

You read Superforecasting, Thinking in Bets, Decisive — then the frameworks die in a read-later app you never reopen. mentatree turns them into skills you invoke on real decisions, right where you already work. Read once, invoke forever.

It grades itself.

Every prediction you make gets a probability and a review date, logged to a private local ledger. When the date comes, mentatree resurfaces it and you mark what happened. Over time it computes your calibration — are your 70%s really 70%?

Each book, a skill.

Distilled frameworks, not summaries — each one acts on your actual decision.

/forecast
Superforecasting · Tetlock

Base rate → decompose → assign a probability. Owns the review + calibration loop.

/bet
Thinking in Bets · Duke

Reframe the choice as a wager — odds, stake, and separating the decision from the luck.

/wrap
Decisive · Heath

Widen options, reality-test, attain distance, prepare to be wrong.

/biascheck
Thinking, Fast and Slow · Kahneman

Name the biases in play and force the outside view. A pure lens — logs nothing.

Read a book worth its own skill? Add it. mentatree is an open library.

Install

mentatree is a Claude Code plugin. Two lines:

# in Claude Code
/plugin marketplace add svenvan945/mentatree
/plugin install mentatree

Zero dependencies. Your decisions stay in a local file (~/.decide) — nothing leaves your machine.
